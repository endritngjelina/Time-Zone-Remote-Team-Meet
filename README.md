# Time-Zone-Remote-Team-Meet

Plan remote team meetings across time zones—fast.

This project is a collection of lightweight **HTML prototypes** for exploring different UI/UX approaches to scheduling meetings for **distributed teams**. Each numbered file represents an iteration with small variations in layout, content structure, or flow. Open any file in your browser to preview.

> **Why this exists:** Remote teams need a simple way to compare time zones and identify reasonable meeting windows. These prototypes test ideas for clarity, speed, and accessibility—without introducing heavy dependencies.

---

## ✨ Features (Prototype Scope)

- **Zero-setup previews:** Just open an `.html` file—no build or server required.
- **Multiple iterations:** Compare versions (`1.html` to `6.html`) to evaluate copy, structure, and potential interactions.
- **Portable & forkable:** Edit in any text editor; easy to share with stakeholders for review.
- **MIT licensed:** Use, modify, and adapt freely for your needs.

> Note: These are **static HTML prototypes**. If you need live time zone logic, user inputs, or calendar integration, see the **Roadmap** below.
